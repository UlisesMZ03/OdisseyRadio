**OdisseyRadio es una aplicación de escritorio construida en C++ usando Qt**, diseñada para experimentar con la reproducción y control de estaciones de radio digitales o archivos de audio.

---

### 🚀 ¿Cómo compilar?

```bash
# Clonar el repositorio
git clone https://github.com/UlisesMZ03/OdisseyRadio.git

# Crear carpeta de build y compilar con CMake
cd OdisseyRadio
mkdir build && cd build
cmake ..
make
./OdisseyRadio
```

> Requiere Qt 6.5+ y CMake instalado en tu sistema
